# URL Shortener
##Description
-This simple Python script utilizes the pyshorteners library to shorten URLs using the TinyURL service.

##Installation
-Ensure you have Python installed on your system. If not, download and install it from Python's official website.
-Install the pyshorteners library by running the following command in your terminal or command prompt:
-Copy code: pip install pyshorteners

##Usage
-Run the script.
-Input the URL you want to shorten when prompted.
-The script will output the shortened URL.
